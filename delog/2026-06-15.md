# 개발일지 — 2026-06-15

## 프로젝트
**토리키즈** — 유아·초등 대상 한글·수학·알파벳 학습 서비스 랜딩 페이지

## 작업 내용

### 1. 리포지토리 세팅
- GitHub 리포지토리 `rest05` 생성 및 로컬 클론
- 기존 토리키즈 프로젝트 파일 복사

### 2. 프로젝트 구조
```
rest05/
├── index.html          # 진입점 (CDN React + Babel)
├── styles.css          # 전체 디자인 시스템
├── app/
│   ├── main.jsx        # 라우팅 및 앱 진입
│   ├── shared.jsx      # 공통 컴포넌트 (Header, Footer 등)
│   ├── home.jsx        # 홈 페이지
│   ├── subpages.jsx    # 서브 페이지 1
│   └── subpages2.jsx   # 서브 페이지 2
└── themes/             # 테마별 CSS 6종
    ├── blue-yellow.css (기본)
    ├── blue-pink.css
    ├── green-yellow.css
    ├── purple-pink.css
    ├── red-yellow.css
    └── teal-coral.css
```

### 3. 기술 스택
- React 18.3.1 (CDN UMD)
- Babel Standalone 7.29.0 (JSX 트랜스파일)
- 빌드 도구 없음 — 순수 HTML + JSX 구성
- 다크모드 / 테마 전환 지원 (localStorage 저장)

### 4. 배포
- GitHub Pages 활성화
- 배포 URL: https://yong275.github.io/rest05/

## 특이사항
- CDN 기반이라 별도 빌드 없이 브라우저에서 직접 실행
- `npx serve`로 로컬 확인 후 푸시 (CORS 방지)
- 로컬과 GitHub Pages 간 미세한 레이아웃 차이 확인 → 스크롤바 너비 및 뷰포트 차이로 추정
